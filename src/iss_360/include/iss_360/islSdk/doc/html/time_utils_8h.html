uint64_t timeGetAppStartTime ( void  )

Get the time that the SDK was started in milliseconds.

Returns
The number of milliseconds between the Unix epoch - 00:00:00 UTC on 1 January 1970 and the starting of the SDK
Note
Add this to the timeMs parameter of events to get the UTC time stamp of the event
