matrix3x3_t *  transform 
)

Calculate the correction matrix and bias This function first calculates a best fit ellipsoid for the list of vector points, built by magCalAddData().

Then a correction matrix and offsets are calculated to transform the ellipsoid into a sphere centered on the x,y,z axis

Parameters
instA pointer to the object
radiusthe raduis of the sphere that points should fit to. Typically this would be the Earths magnetic field strength 54uT
transformA pointer to a matrix3x3_t where the matrix will be written
biasA pointer to a vector_t where the bias will be written
Returns
  • TRUE if succeeded
  • FALSE transform and bias were not populated
