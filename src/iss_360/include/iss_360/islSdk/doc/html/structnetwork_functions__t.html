◆ openSocket

socket_t *(* openSocket) (bool_t isTcp, bool_t isServer, uint32_t ipAddress, uint16_t port)

Function pointer to open a socket @Param tcp - TRUE open a TCP socket.

  • FALSE open a UDP socket @Param isServer - TRUE if the socket is a server, (listen for data before sending)
    • FALSE if the socket is a client, (send data first) @Param ipAddress - In server TCP mode: the IP Address to listen for a connection from.
      • In client TCP mode: the IP Address to connect and send data to.
      • In server UDP mode: the IP Address to allow data from. 0 = any IP
      • In client UDP mode: not used. Specify the IP address when sending data @Param port - In server TCP mode: the port to listen for a connection from.

In client TCP mode: the port to connect and send data to.

  • In server UDP mode: the port to allow data from. 0 = any port
  • In client UDP mode: not used. Specify the port when sending data
    Returns
    Pointer to an object
